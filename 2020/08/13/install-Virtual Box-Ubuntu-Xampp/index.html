安裝Virtual Box與Ubuntu與Xampp

安裝Virtual Box

下載Virtual Box（含 Extension Pack）

https://www.virtualbox.org/

  • VirtualBox免費的模擬器,可以不用砍掉window的作業系統,就可以使用Linux的作業系統

安裝Virtual Box 接著安裝 Oracle_VM_VirtualBox_Extension_Pack

開啟VirtualBox新增一個模擬器位置給Ubuntu

本機分多少RAM(通常一半)

立即建立

VDI

動態分配

容量最大可以用到硬碟多少G

建立好再啟動並安裝Ubuntu

安裝Ubuntu

下載Ubuntu 20.04 64位元桌面版

http://www.ubuntu-tw.org/modules/tinyd0/

  • 作業系統「Ubuntu 20」是以桌面應用為主的Linux發行版(有很多不同的發行版,此為其一)

安裝Ubuntu





安裝好作業系統Ubuntu(帳:kaimin,密:生日6碼)

設定全螢幕


跑好按「enter鍵」,然後關機

設定Virtual Box,然後啟動


調整為全螢幕模式
「Right ctrl + F」可切換Linux的視窗<->全螢幕,或以下

終端機加入喜好(所有東西都要用下指令)

安裝Xampp

下載Xampp (Linux版 PHP 7.4.8)

https://www.apachefriends.org/zh_tw/download.html

  • XAMPP是最流行的PHP開發環境。
  • XAMPP是完全免費且易於安裝的Apache發行版本，其中包含MariaDB、PHP和Perl。別人整理好的一大包的伺服器。
  • XAMPP開放源碼套件的設置讓安裝和使用出奇容易。
  • MariaDB跟MySQL一模一樣
    (因為MySQL被甲骨文收購,公司怕從開源變閉源(ex:Java)會有挨告風險,於是再創一個開源)

在Ubuntu用firefox下載Xampp

用終端機安裝Xampp

用終端機 Xampp start & stop

在firefox搜尋欄打「localhost」
進入phpMyAdmin

  • phpMyAdmin是一個以PHP為基礎,以Web-Base方式架構在網站主機上的MySQL的資料庫管理工具
